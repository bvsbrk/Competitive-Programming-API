Competitive Programming API

Capable of fetching contests from:
  • Hackerrank
  • Codechef
  • Codeforces
  • Hackerearth
  • CS Academy
  • Top coder
Fetching contests data:
Contests can be fetched by passing the name of the site as follows:
https://api-coders-kit.herokuapp.com/site_name/contests
Example: Contests of codechef can be fetched by https://api-coders-kit.herokuapp.com/codechef/contests
Response:

Response as shown above will be sent but the only difference is that each and every live and future and past contests will be sent. It will be in the pure json format without any style applied.The start and end time will be shown in UNIX time format. It can be easily converted to required time zone.
